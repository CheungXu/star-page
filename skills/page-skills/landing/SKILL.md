# 产品落地页专项制作指南

为"营销转化"场景生成有冲击力、说服力强、引导清晰的单页落地页。

## 信息架构（推荐模块顺序）

1. 首屏 Hero：一句强价值主张（大标题）+ 副标题说明 + 主 CTA 按钮 + 视觉主体（产品图/抽象图形/渐变）。
2. 痛点/价值：用户为什么需要它，3 点左右。
3. 核心功能/卖点：3-6 个卡片，每个含图标、标题、一句说明。
4. 工作原理/使用步骤：1-2-3 步骤化展示。
5. 社会证明：用户评价、数据指标（用户数/好评率）、合作 logo 墙。
6. 价格/方案（如适用）：2-3 档对比，突出推荐档。
7. 常见问题 FAQ：折叠手风琴。
8. 结尾 CTA + 页脚：再次行动号召 + 版权/链接。

## 视觉与排版

- 首屏要有强对比和呼吸感：大标题、充足留白、清晰的主按钮。
- 一个鲜明的品牌主色 + 强调色用于 CTA；可用渐变/光晕营造现代氛围。
- 卡片化呈现卖点，统一圆角、阴影与间距，建立视觉节奏。
- 主 CTA 在全页多次出现（首屏、价格区、结尾），样式一致且最显眼。

## 交互（增强转化）

- 滚动出现动画（fade/slide-up）逐步揭示模块。
- 数字指标可用计数器动画；FAQ 用折叠；功能区可用 tab 切换。
- CTA hover 有明显反馈；保证禁用 JS 时主体内容与按钮仍可见可读。

## 易错点

- 价值主张要具体、可感知，避免空泛口号。
- 不要让卖点过多导致首屏失焦；先冲击、再展开。
- 移动端首屏 CTA 必须可见，模块单栏堆叠、间距合理。
