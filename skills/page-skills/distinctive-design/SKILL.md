# 强风格设计专项指南

目标：做出有明确美学主张、令人过目不忘的页面，坚决避免"通用 AI 观感"。改编自 Anthropic frontend-design 理念，落到本项目的单文件自包含 HTML 展示页。

## 先定方向，再动手

- 先承诺一个**大胆而明确的美学方向**（任选其一并贯彻到底）：极简克制、极繁张力、复古未来、自然有机、奢华精致、玩味童趣、杂志编辑、粗野主义、装饰几何、柔和粉彩、工业实用……
- 关键是**意图清晰**而非堆砌强度：大胆的极繁与克制的极简都成立，前提是从头到尾一以贯之。
- 想清楚"这个页面最让人记住的一处是什么"，并把它做足。

## 视觉要点

- **字体**：拒绝 Inter / Roboto / Arial / 系统默认这类通用字体的平庸观感；用有性格的标题字 + 克制易读的正文字搭配。优先用字号、字重、字距、行高、排版编排制造张力；如需 Web 字体只走可信 CDN 并带可靠回退。
- **配色**：一套连贯的主色 + 锐利点缀色，强于平均分布的胆怯配色；用 CSS 变量统一管理。避开烂大街的"白底紫渐变"。
- **动效**：聚焦高价值瞬间——一次精心编排的入场（stagger 渐次显现）比满屏零碎微交互更出彩；配合滚动出现、出人意料的 hover。优先纯 CSS，尊重 `prefers-reduced-motion`。
- **空间构成**：尝试非常规布局——不对称、重叠、对角线流向、打破网格的元素；大面积留白或受控的高密度二选一并做到位。
- **背景与质感**：用渐变网格、噪点、几何图案、层叠透明、戏剧化阴影、颗粒覆盖等营造氛围与层次，而不是默认大色块平涂；优先用 CSS/SVG 实现以保持自包含。

## 红线

- 不要出现通用 AI 审美：滥用通用字体、白底紫渐变、千篇一律的卡片栅格与可预测排版。
- 复杂度要匹配愿景：极繁方向就把动效与细节做满，极简方向就在间距、字体、微差上做到极致克制。
- 每次都应不同：在明暗主题、字体、风格上有意区分，不要每次都收敛到同一套选择。
