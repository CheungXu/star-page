# 简历页专项制作指南

为"个人履历/求职"场景生成专业、克制、易读的单页简历。

## 信息架构（推荐模块顺序）

1. 顶部抬头：姓名（最大字号）、一句话定位/职位、联系方式（邮箱、电话、城市、主页/GitHub）。
2. 个人简介：2-4 句概述优势与求职意向。
3. 工作/项目经历：按时间倒序，每条含「职位 · 公司 · 起止时间」+ 2-4 条要点（动词开头、量化成果）。
4. 教育背景：学校、专业、学位、时间。
5. 技能：分组展示，可用标签或带等级的进度条。
6. 可选：获奖、证书、语言、自我评价。

## 视觉与排版

- 优先单栏或"左侧栏 + 右主栏"两栏布局；左栏放联系方式/技能，右栏放经历。
- 配色克制：一种主色（建议沉稳的蓝/墨绿/深灰）+ 中性灰，避免花哨渐变。
- 字号层级清晰：姓名 > 模块标题 > 条目标题 > 正文；行高宽松（1.5 左右）。
- 用细分隔线、左侧时间线竖线或小圆点表达经历的时间顺序。
- 适度留白，保证打印/导出 PDF 时排版整洁（可加 `@media print` 优化）。

## 交互（克制）

- 进入时轻微淡入即可；技能进度条可在出现时做一次性增长动画。
- 不要做轮播、弹窗等与简历无关的花哨交互。

## 易错点

- 不要堆砌长段落；经历用要点而非整段叙述。
- 时间、公司、职位三要素要对齐、统一格式。
- 移动端要单栏堆叠，联系方式可点击（mailto/tel）。
