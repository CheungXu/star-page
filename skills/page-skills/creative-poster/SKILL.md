# 创意海报专项指南

目标：做出可当作艺术品/封面来欣赏的单页视觉。改编自 Anthropic canvas-design 的"设计哲学先行"思路，落到本项目的单文件自包含 HTML（用 CSS/SVG 表达，不依赖外部图片与网络）。

## 设计哲学先行

- 先为这件作品确立一个**视觉哲学/美学主张**（给它起个 1-2 个词的名字，如"混凝土诗""色域语言""留白冥想"），再据此表达。
- 让信息活在设计里，而非段落里：成品约 **90% 视觉 / 10% 必要文字**。

## 表达要点

- **文字即视觉元素**：以巨大的主标题词 + 极小的标签点缀，绝不大段成段落；文字本身参与构图（超大字、字距、排版编排）。
- **构图与工艺**：像顶尖设计师打磨过无数小时——构图、间距、配色、对齐都体现专家级精致度；克制而精准。
- **色彩**：有意而连贯的有限调色板，强调整体气质，而非花哨多色。
- **形式**：用 CSS/SVG 的几何形、色块、重复图案、渐变、层叠制造强视觉；可借助重复与系统化标记营造"像某种学科图鉴"的质感。
- **版面安全**：所有文字与图形必须完整落在页面内、有充足边距，**不溢出、不裁切、不重叠**；这是专业执行的底线。
- **单屏优先**：默认面向单屏/单页的强视觉呈现；如需多页/分屏，各屏沿同一哲学但各有变奏。

## 精修

- 二次打磨时**不要靠加更多图形**，而是让既有构图更紧致、更统一、更像一件作品；先问"怎样让现有元素更像艺术"，再决定是否新增。
- 始终保持成熟、设计导向，避免卡通化或业余感。
