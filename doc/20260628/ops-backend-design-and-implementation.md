# 星页运营后台 · 设计与实施记录

## 一、背景与目标

财务后台已能回答「赚不赚钱」，但回答不了「产品健不健康、用户爱不爱用、增长从哪来」。本次新建**运营后台**，与财务后台互补、互为入口，让管理者无需翻库、无需找研发，即可随时洞察产品状态：

- 看得到**实时动态**（谁在用、在生成什么、有没有异常）。
- 看得清**日 / 周 / 月趋势**（用户量、DAU/WAU/MAU、留存、转化）。
- 看得到**详细用户使用情况**与**具体 Case**（能下钻到单个用户画像、单次生成的完整过程）。

## 二、北极星指标（North Star）

我们没有用泛泛的「DAU」或「注册数」当北极星，而是定义了更贴合本产品价值链的复合指标：

> **WAGP = 最近 7 天「成功生成且产生了外部访问」的页面数**
> （Weekly Activated Generated Pages with external visits）

它一个指标同时约束三件事：用户**真的在创作**（生成成功）、产物**有质量**（值得被看）、并且**对外传播**（被作者之外的人访问）。北极星上去了，说明产品的核心价值闭环（创作→分享→拉新）在转动；只刷注册或只刷生成都骗不过它。

## 三、指标体系（11 个域）

| 域 | 回答的问题 | 代表指标 |
| --- | --- | --- |
| 实时 | 此刻产品在发生什么 | 在线数、进行中/排队任务、近 1h 失败、今日累计 |
| 增长 | 用户规模在涨吗 | 新增注册/匿名、累计注册、DAU/WAU/MAU |
| 激活 | 新用户用起来了吗 | 注册后 24h 内成功生成率 |
| 留存 | 用户回来吗 | 登录留存、创作留存（D1/D3/D7/D14 矩阵） |
| 参与 | 用得有多深 | 续写率、文件上传率、人均生成、模型/技能分布 |
| 质量 | 体验稳不稳 | 成功率、耗时 P50/P90/P99、各模型成功率、失败原因 |
| 传播 | 能自增长吗 | 页面 PV/UV、分享传播比、热门页 TOP |
| 商业化 | 钱怎么来 | 充值金额、新增付费用户、积分消耗（明细以财务后台为准） |
| 漏斗 | 哪一步在漏 | 落地→输入→点击生成→生成成功→注册→首充 |
| 用户明细 | 单个用户是谁 | 画像（生成/成功率、会话/页面、充值/消费、最近会话） |
| Case | 这次生成发生了什么 | 完整 prompt/输出/用量/SSE 事件流/访问数 |

> 独到处：①北极星用「外部访问」把传播力写进核心指标；②留存拆「登录留存」与「创作留存」——后者更能反映真实复用价值；③续写率被当作高价值正向信号（用户在认真打磨）；④漏斗与质量分别从前端意图和后端结果两端度量，互相印证。

## 四、数据架构

采用**两层架构**，兼顾实时新鲜与历史快查：

- **明细层**：生成页访问日志、前端漏斗事件。逐条记录，支撑实时看板与传播/质量下钻。隐私上 **IP 仅存哈希**（HMAC），不落原文。
- **聚合层**：每天把明细与业务数据预计算成「逐日指标 / 留存 cohort / 漏斗」快照。趋势、总览、留存、漏斗读快照，秒级返回且口径稳定。

聚合任务由 systemd 定时器驱动：每小时补当天、每日凌晨收口昨日并重算留存，可断点补漏、可安全重跑回填历史。

## 五、交付范围

- 后端：5 张数据表 + 采集接口（公开埋点，含白名单与限频）+ 13 个管理端只读接口 + 离线聚合引擎与 CLI + 定时器模板。
- 前端：`/ops` 运营后台单页，九个 Tab（实时看板 / 增长 / 留存 / 参与 / 质量 / 传播 / 商业化 / 用户明细 / Case 查看），实时看板 15 秒自动刷新，图表为**零依赖 SVG**（折线 / 条形 / 留存热力），用户与 Case 支持搜索筛选与弹窗下钻。与财务后台 `/admin` 互加入口，复用现有管理员鉴权。

## 六、上线与验证

- 数据库迁移 `016_analytics.sql` 已建表；后端新接口已注册并重启生效（公开埋点返回 204、管理端无权限正确 401）。
- 以真实库数据全量走查：实时看板、九个 Tab、用户画像弹窗、Case 详情弹窗（含 SSE 事件流）均正常渲染，趋势/留存/分布图表数据正确。
- 前端埋点经 `sendBeacon` 上报，验证落库且 IP 已哈希。

## 七、后续可迭代

- 激活/漏斗的「前端意图」与「后端结果」目前分别度量，后续可通过统一 `client_session_id` 串联做端到端归因。
- 聚合可随数据量增长引入分区表或物化视图；实时看板可从轮询升级为推送。
- 可增加自定义看板、指标告警（如失败率突增、北极星周环比下滑）与导出。
